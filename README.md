# controle-nivel-agua
# Controle de Níveis de Reservatório

Sistema de monitoramento desenvolvido em Python para simular alertas de níveis de água utilizando cores no terminal.

## 🛠️ Tecnologias
* Python 3
* Biblioteca Colorama

## 🚀 Como executar
1. Instale a biblioteca necessária:
   pip install colorama
2. Execute o script:
   python monitoramento.py
